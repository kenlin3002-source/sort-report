📚 演算法概念

快速排序是基於分而治之 (Divide and Conquer) 的演算法。核心是「分割 (Partition)」操作：

  • 選擇基準 (Pivot)：從陣列中挑選一個元素作為基準值 (通常選最後一個)。
  • 分割 (Partition)：重新排列陣列，讓所有比 Pivot 小的元素移到其左邊，比 Pivot 大的元素移到右邊。此時 Pivot 就會處於最終正確的位置！
  • 遞迴 (Recursion)：分別對 Pivot 左邊和右邊的子陣列重複上述步驟，直到整個陣列排序完成。
💡 特點： In-place 排序，不需要額外的大量記憶體空間（除了遞迴呼叫的堆疊空間）。在多數情況下，它是執行速度最快的排序演算法之一。

⏱️ 複雜度分析

平均時間複雜度
O(n log n)
最壞時間複雜度
O(n²)

💻 程式碼對應

function quickSort(arr, low, high) { if (low < high) { // 取得分割點 (Pivot 就位) let pi = partition(arr, low, high); // 遞迴排序左右子陣列 quickSort(arr, low, pi - 1); quickSort(arr, pi + 1, high); } } function partition(arr, low, high) { let pivot = arr[high]; // 選擇最後一個元素作基準 let i = low - 1; // 較小元素的索引 for (let j = low; j < high; j++) { if (arr[j] < pivot) { i++; [arr[i], arr[j]] = [arr[j], arr[i]]; // 交換 } } // 將 pivot 放到正確的中間位置 [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]]; return i + 1; }
def quick_sort(arr, low, high): if low < high: pi = partition(arr, low, high) quick_sort(arr, low, pi - 1) quick_sort(arr, pi + 1, high) def partition(arr, low, high): pivot = arr[high] i = low - 1 for j in range(low, high): if arr[j] < pivot: i += 1 arr[i], arr[j] = arr[j], arr[i] arr[i + 1], arr[high] = arr[high], arr[i + 1] return i + 1
void quickSort(vector<int>& arr, int low, int high) { if (low < high) { int pi = partition(arr, low, high); quickSort(arr, low, pi - 1); quickSort(arr, pi + 1, high); } } int partition(vector<int>& arr, int low, int high) { int pivot = arr[high]; int i = low - 1; for (int j = low; j < high; j++) { if (arr[j] < pivot) { i++; swap(arr[i], arr[j]); } } swap(arr[i + 1], arr[high]); return i + 1; }
